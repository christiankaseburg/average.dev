# Coding Standards & Formatting Constraints

## 🛠️ Linter & Formatter Enforcement
Do not guess formatting standards or invent styling conventions. Before finalizing file writes, run the project formatter via the integrated terminal surface:
- Target Command: `npx nx run-many -t lint --fix`

## 📝 Syntax Rules
- Always prioritize explicit typing over implicit type assumptions.
- Avoid nesting logical execution blocks deeper than three levels.
